# ☕ Pranav's Cafe

Welcome to the digital storefront for **Pranav's Cafe**! This project is a modern, hyper-realistic, and highly interactive web application built with a premium dark-themed aesthetic, focusing on an unforgettable user experience. 

## ✨ Key Features

- **Hyper-Realistic 3D Coffee Cup**: Powered by **Three.js** and `RoomEnvironment` HDR studio lighting, bringing a fully sculpted ceramic coffee cup to life right on the homepage.
- **Dynamic Fog Atmosphere**: Utilizing **Vanta.js** to create a mesmerizing, swirling fog background that perfectly matches the cafe's rich bronze and olive color palette.
- **Glassmorphism UI**: Beautiful frosted-glass forms spanning the application that let the background effects shine through elegantly.
- **Interactive Ordering System**: A seamlessly integrated, dynamic shopping cart calculator that live-updates the total estimate without any page reloading. 
- **Fully Responsive**: Designed to look stunning and perform beautifully on both desktop and mobile devices.

## 🛠️ Technology Stack

- **HTML5** & Semantic Markup
- **Vanilla CSS3** (Custom Properties, Flexbox, Grid, Glassmorphism Backdrop-Filters)
- **Vanilla JavaScript** (ES6 Modules)
- **Three.js** (3D WebGL Rendering)
- **Vanta.js** (Interactive Fog Canvas Background)

## 🚀 Getting Started

To run this project locally, simply clone the repository and start a simple static server.

### Prerequisites
You can run this project with Python’s built-in HTTP server or any extension like VSCode Live Server.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/pranavs-cafe.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pranavs-cafe
   ```
3. Start a local development server:
   ```bash
   python3 -m http.server 8000
   ```
4. Open your browser and visit: `http://localhost:8000`
